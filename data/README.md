# Data availability

The processed expression matrices required to run the analysis notebooks are available from Zenodo:

> Zeballos S, et al. Processed expression matrices associated with “Combining transcriptomic resolutions and machine learning strategies uncovers new OXPHOS genes in Caenorhabditis elegans.” Zenodo. DOI: [10.5281/zenodo.21376210]

Download the following files and place them inside the `data/` directory:

```text
data/
├── adult_expression_time_adjusted.csv
├── boeck.csv
└── embryo_expression_time.csv
```

The matrices were derived from publicly available datasets reported by Boeck et al. (2016), Packer et al. (2019), and Ghaddar et al. (2023). Users should cite both the Zenodo dataset and the corresponding original publication when reusing these processed matrices.
